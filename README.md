##1. 개요</br>
Project PIA는 유니티를 처음 배울 때부터 습득한 지식과 유용한 함수들을 JangEngine이라는 개인 라이브러리로 통합하여 이를 토대로 여러 장르의 게임을 만들어 묶어놓은 게임팩입니다.</br>
총 7가지의 게임으로 구성돼 있으며 옆 그림과 같은 게임 진행 방식을 취하고 있습니다.</br>
